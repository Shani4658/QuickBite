# 🍔 QuickBite

QuickBite is a modern food delivery web application inspired by popular platforms like Swiggy and Zomato. Built using React.js and Tailwind CSS, the application provides a responsive and interactive user experience for browsing restaurants, viewing menus, searching food items, and managing carts.

---

# 🚀 Features

## 🍽️ Restaurant Browsing

* Browse restaurants dynamically
* Restaurant cards with ratings and details
* Restaurant menu pages
* Category-wise food items
* Responsive restaurant listing UI

## 🔍 Search & Filtering

* Search restaurants by name
* Dynamic filtering functionality
* Fast and smooth user experience

## 🛒 Cart Management

* Add items to cart
* Remove items from cart
* Dynamic cart updates
* Redux-based state management

## ⚡ User Interface

* Fully responsive design
* Modern food delivery UI
* Shimmer loading effect
* Reusable React components
* Smooth navigation using React Router

## 🧪 Testing Support

* Unit testing with Jest
* React Testing Library integration
* Component-level testing

# 🧠 Application Workflow

1. User lands on the homepage
2. Restaurants are fetched dynamically
3. Users can search restaurants and food items
4. Clicking a restaurant opens its menu page
5. Users can add or remove items from the cart
6. Cart updates dynamically using Redux state management
